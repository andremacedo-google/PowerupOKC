# Catálogo de Data Objects (Objetos de Dados)

Este arquivo de `index.md` atua como o portal de entrada para o ecossistema de dados da **PowerUp OKC**, dividindo o acervo conceitual de informações em dois grandes formatos de dados:

## 1. Dados Estruturados (Subtítulo: Structured)
Dicionários lógicos, esquemas e registros que trafegam entre sistemas transacionais core de TI/TO (IDs DO-101 a DO-181).
*   [index-data-objects-structured.md](Dados Estruturados) - Detalhamento das tabelas e barramentos estruturados.

## 2. Dados Não Estruturados (Subtítulo: Unstructured)
Acervos documentais, leis federais, runbooks operacionais, resoluções ANEEL e contratos estratégicos (IDs DO-201 a DO-223) que servem como bases semânticas confiáveis para o grounding de agentes RAG.
*   [index-data-objects-unstructured.md](Dados Não Estruturados) - Detalhamento das bases de conhecimento do acervo documental.
